# Web portfolio

## Installation

- Install Node.js

	https://nodejs.org/en/

- Install Gulp

```
npm install gulp
```

- Install gulp-sass

```
npm install gulp-sass
```

## Kirby plugins

- Add the "gallery" plugin to generate galleries

```
https://github.com/TimOetting/kirby-gallery
```

Copy it in site > fileds

Now inside the blueprint you can add a gallery by adding this field:

```
mygallery:
    label: My Gallery
    type: gallery
```
